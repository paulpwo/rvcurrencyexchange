== Description ==

A new WordPress plugin generated with Yeoman.

yo @wptools/plugin

yo @wptools/plugin:submenupage



== Installation ==

=== From within WordPress ===

1. Visit 'Plugins > Add New'
1. Search for 'Rvcurrency Exchange'
1. Activate Rvcurrency Exchange from your Plugins page.
1. Go to "after activation" below.

=== Manually ===

1. Upload the `wordpress-seo` folder to the `/wp-content/plugins/` directory
1. Activate the Rvcurrency Exchange plugin through the 'Plugins' menu in WordPress
1. Go to "after activation" below.
